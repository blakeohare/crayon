Game.GameWindow

Represents the game window or the browser frame or mobile screen. Manages the size of the screen, frame rate, and user input. For drawing graphical information to the screen, see the Graphics2D library.

new GameWindow(title, fps, width, height, screenWidth, screenHeight)

new GameWindow(title, fps, width, height, screenWidth, screenHeight)
Instantiates a new game window. For most platforms this will open a window or initialize the rendering surface (e.g. JavaScript's canvas). Furthermore, the screenWidth and screenHeight parameters may be ignored in platforms that don't have a notion of a dynamically sized window (such as mobile with a fixed-size strings). Creation of the window will also automatically show the window (i.e. there are no .show() or .hide() methods).

Arguments

NameType
titlestringThe title to appear in the window (if supported by the platform). Certain platforms require a name before the VM has a chance to run (such as the setting the title of the HTML host page or the name of a mobile app. For these, use the title field in the build file.
fpsfloatThe desired frames per second for the game.
widthintegerLogical width of the window.
heightintegerLogical height of the window.
screenWidthintegerActual width of the window. The image will be stretched to fit this size.
screenHeightintegerActual height of the window. The image will be stretched to fit this size.

Game.KeyEvent

Event that gets fired when a keyboard key is pressed or released.

keyEvent.key

The key that is pressed.

keyEvent.down

True if the key was pressed down.

Game.MouseClickEvent

Event that gets fired when a mouse button is pressed or released.

mouseClickEvent.button

The mouse button that was pressed or released.

mouseClickEvent.down

True if the mouse was pressed down. False if it is released.

Game.MouseEvent

Event that gets fired when a mouse event occurs. This is an abstract base class for other more specific mouse events.

mouseEvent.x

X coordinate for the mouse event.

mouseEvent.y

Y coordinate for the mouse event.

Game.MouseMoveEvent

Event that gets fired when the mouse moves. Does not have any special fields since the X and Y values are in the base class.

Game.MouseScrollEvent

Event that gets fired when the scroll wheel is scrolled.

mouseScrollEvent.amount

The amount that was scrolled.

Game.QuitEvent

Event that gets fired when a quit attempt has occured, such clicking the close button or pressing Alt+F4.

quitEvent.quitType

Enum value describing how this quit event was invoked.
