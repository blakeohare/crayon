enum TextEncoding[link]

Available text encodings for files. Used by FileIO functions.

NameDescription
UTF8

The UTF-8 encoding without a Byte-Order-Mark.

UTF8_BOM

The UTF-8 encoding with the Byte-Order-Mark.

UTF16

The UTF-16 encoding.

UTF32

The UTF-32 encoding.

ASCII

Basic ASCII encoding.
